# PermitPulse outreach target segmentation

## ADU builders
- Likely need permit-history clarity during takeover, quote cleanup, or owner education.
- Best angle: practical support for one address when the project story is fuzzy.

## Permit expediters
- Likely need a backup research layer for routing confusion or messy property history.
- Best angle: not a replacement for expediting, a support tool for permit-history cleanup.

## Architects and designers
- Likely need a clean permit-history baseline before design intake, remodel strategy, or takeover work.
- Best angle: summarize the file before concept or coordination time gets burned.

## Investor-friendly agents
- Likely need faster property-level diligence for buyers or value-add listings.
- Best angle: clearer permit history before a recommendation, offer, or listing strategy.

## Homeowners and owner-builders
- Likely need clarity before refinance, sale prep, inherited project handoff, or ADU cleanup.
- Best angle: one readable property-level report instead of piecing together the permit trail alone.
